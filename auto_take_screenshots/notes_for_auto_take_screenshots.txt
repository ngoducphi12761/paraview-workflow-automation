Place these 3 files

	1. auto_take_screenshots.py

	2. auto_take_screenshots_pv562.py

	3. auto_take_screenshots_pv5101.py

into this path:
C:\Users\phiducngo\AppData\Roaming\ParaView\Macros\

Then open paraview 5.6.2 or 5.10.1 then choose auto_take_screenshots to export screenshots (auto_take_screenshots script will automatically detect which paraview version has been currently using)

This automation only works on:

	a) PV 5.10.1 – works on both a remote Linux server and a client Windows 11 machine.

	b) PV 5.6.2 – works only on a client Windows 11 machine.
